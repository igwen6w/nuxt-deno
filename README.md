## 项目设置

查看 [Nuxt 文档](https://nuxt.com/docs/getting-started/introduction) 了解更多。

### 安装依赖

```bash
deno cache --reload deps.ts
```

### 开发服务器

启动开发服务器 `http://localhost:3000`:

```bash
deno task dev
```

### 生产环境

构建生产环境应用:

```bash
deno task build
```

本地预览生产构建:

```bash
deno task preview
```

查看 [部署文档](https://nuxt.com/docs/getting-started/deployment) 获取更多信息。
