## 项目设置

查看 [Nuxt 文档](https://nuxt.com/docs/getting-started/introduction) 了解更多。

### 安装依赖

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

### 开发服务器

启动开发服务器 `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

### 生产环境

构建生产环境应用:

```bash
# npm
npm run build

# pnpm
pnpm build

# yarn
yarn build

# bun
bun run build
```

本地预览生产构建:

```bash
# npm
npm run preview

# pnpm
pnpm preview

# yarn
yarn preview

# bun
bun run preview
```

查看 [部署文档](https://nuxt.com/docs/getting-started/deployment) 获取更多信息。
